Looping in Ruby and JavaScript

This is a comparison between looping in Ruby and JavaScript. First let's go over JavaScript, because it's in some sense simpler with fewer options.

JavaScript: for

for (var i=0; i<10; i++) {
  console.log(i);
}

JavaScript: while

var i = 0;
while (i<10) {
  console.log(i);
  i++;
}

JavaScript: for in

var obj = {a:1, b:2, c:3};
for (var prop in obj) {
  console.log(obj[prop]);
}

Now let's look over the different loops Ruby has to offer. The fundamental difference is that JavaScript isn't designed to loop over objects, whereas Ruby can nothing but loop over objects.

Ruby: for in

for i in 0...10
  puts i
end

Ruby: each

(0...10).each do |i|
  puts i
end

Ruby: while

i = 0
while i < 10 do {
  puts i
  i += 1
end

Ruby: until

i = 0
until i >= 10 do {
  puts i
  i += 1
end

Besides these basic loops, Ruby provides multiple methods for enumerables, like the each method. These methods allow you to modify, group, or create new enumerables using a method instead of writing out a complete loop structure.
